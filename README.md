# Simple Gas Flow Calculator

This is rough calculator to calculate flow rate of gases through known pipe and orifice.

### How To Run Locally
Run from terminal :
```shell
$ cd gas-flow-calculator
$ npm run dev
```
or 
```shell
$ npm run start
```
